# 42 London: Discovery Piscine (Python & Bash)
### February 2026 | Peer-to-Peer Technical Learning

## 🚀 Overview
This repository contains my solutions and scripts from the **42 London Discovery Piscine**. This intensive, week-long program focused on fundamental Python programming, Bash scripting, and the critical interaction between the terminal environment and Python's `sys` module. 

The learning methodology at 42 emphasizes peer-to-peer collaboration and solving technical challenges through self-directed research and experimentation.

---

## 🧠 Key Learning Milestones

### **Module 4: Basic Types & Objects**
* **Data Casting:** Managed user inputs by converting strings to integers (`int()`) and floats (`float()`) for mathematical operations.
* **Error Handling:** Implemented `try-except` blocks to handle `ValueError` exceptions during input sanitization.
* **Built-in Methods:** Utilized string methods like `.upper()` and `.swapcase()` to manipulate text data programmatically.

### **Module 5: Lists & System Parameters**
* **List Comprehensions:** Developed efficient logic for array transformations using one-line comprehensions with conditional filtering.
* **Data Structures:** Leveraged Python `Sets` to handle and remove duplicate values from processed arrays automatically.
* **CLI Interactions:** Used the `sys` library and `sys.argv` to create scripts that interpret command-line parameters.
* **Shell Logic:** Debugged terminal-specific issues such as "History Expansion" (handling special characters like `!` and `?` in `zsh`) and proper quoting techniques.
